Was ist JWT?

Login Flow

  • User sendet Login-Daten
  • Server gibt JWT zurück
  • Client speichert Token
  • Token wird bei jeder Anfrage mitgesendet

Vorteile / Nachteile

✅ Vorteile: Stateless, kompakt, API-freundlich

❌ Nachteile: Kein Logout, Sicherheitsrisiken
